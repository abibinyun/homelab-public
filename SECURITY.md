## Reporting a Vulnerability

**Jangan buat GitHub Issue untuk vulnerability.**

Kirim email ke: **jobs.muhammadbilal@gmail.com**

Sertakan:
- Deskripsi vulnerability
- Langkah reproduksi
- Dampak potensial

Response dalam 48 jam. Patch akan dirilis secepat mungkin.
